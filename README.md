# Sistemas Operativos 2026

Proyecto Django con configuración inicial y app de bienvenida.

## Pasos realizados

### 1. Configuración de SSH
- Se generó una clave SSH con el algoritmo ED25519
- Se agregó la clave pública a GitHub

### 2. Configuración del repositorio
- Se creó el repositorio en GitHub
- Se inicializó git localmente y se conectó al repositorio remoto

### 3. Entorno virtual
- Se creó un entorno virtual con python3
- Se instaló Django
- Se generó el archivo requirements.txt

### 4. Proyecto Django
- Se creó el proyecto con django-admin
- Se creó la app principal
- Se configuró una vista que muestra "Hola Mundo"
